== Description ==
Free IFSC Code Lookup Widget for WordPress.
This plugin embeds an iframe-based IFSC lookup tool from PMetroMart.
No data is collected or stored by the plugin.

== Installation ==
1. Upload the plugin folder to /wp-content/plugins/
2. Activate the plugin
3. Use shortcode [ifsc_widget]
